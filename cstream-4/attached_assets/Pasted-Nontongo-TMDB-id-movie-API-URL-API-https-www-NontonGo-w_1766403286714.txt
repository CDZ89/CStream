Endpoint
https://111movies.com/movie/{id}
Valid parameters:
- {id} required - from imdb.com or themoviedb.com, imdb id must have tt prefix.

Examples
https://111movies.com/movie/tt6263850

https://111movies.com/movie/533535

Endpoint
https://111movies.com/tv/{id}/{season}/{episode}
Valid parameters:
- {id} required - from imdb.com or themoviedb.com, imdb id must have tt prefix.

- {season} required - the eason number

- {episode} required - the episode number

Examples
https://111movies.com/tv/tt30217403/1/5

https://111movies.com/tv/240411/1/5

lecteur moviesapi :

Endpoint
https://moviesapi.to/movie/$id
Valid parameters:
- $id required: Can be a tmdbid or imdbid.

Examples
https://moviesapi.club/movie/385687 (tmdbid)

https://moviesapi.club/movie/tt5433140 (imdbid)

Endpoint
https://moviesapi.to/tv/$id-$season-$episode
Valid parameters:
- $id required: Can be a tmdbid or imdbid.

- $season required: The season number.

- $episode required: The episode number.

Examples
https://moviesapi.club/tv/1399-1-3 (tmdbid for Season 1, Episode 3)

https://moviesapi.club/tv/tt0944947-2-5 (imdbid for Season 2, Episode 5)

Endpoint
https://moviesapi.to/api/discover/movie
Query Parameters:
page: The page number. Default: 1.
resultsPerPage: Number of results per page. Default: 20.
ordering: Sorting field (id, year, upload_date).
direction: Sort direction (asc or desc).
query: Search for a movie by title. Minimum 3 characters.
year: Filter movies by release year.
imdb_id: Search by IMDb ID.
Example Request:
https://moviesapi.to/api/discover/movie?direction=desc&page=1

Endpoint
https://moviesapi.to/api/discover/tv
Query Parameters:
page: The page number. Default: 1.
resultsPerPage: Number of results per page. Default: 20.
ordering: Sorting field (id, year, last_upload_date).
direction: Sort direction (asc or desc).
query: Search for a TV show by title. Minimum 3 characters.
Example Request:
https://moviesapi.to/api/discover/tv?direction=desc&page=1
