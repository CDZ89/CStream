### Instruction pour l’agent: ajout du bandeau explicatif au-dessus de tous les lecteurs

Ajoute un bandeau explicatif avec l’icône nuage au-dessus des lecteurs sur toutes les pages de lecteur (Movie Detail, TV Detail, embeds, variantes). Il doit être visible, premium, responsive, et ne pas modifier le sandbox des iframes.

---

#### Spécifications de placement et portée

- **Emplacement:** Directement au-dessus du conteneur du lecteur (player wrapper), dans toutes les pages et templates de lecteur.
- **Portée:** S’applique à tous les affichages (toutes les variantes de lecteurs, versions desktop/mobile/tablet).
- **Sandbox:** Ne pas modifier les attributs sandbox des iframes existants.
- **Internationalisation:** Le texte doit être i18n-ready (clé, ex. `player.cloud_hint`).
- **Accessibilité:** Icône et texte lisibles, contraste suffisant, aria-label sur le nuage.

---

#### Contenu du bandeau

- **Texte:** “Si tu vois ce nuage dans le lecteur ci‑dessous, clique dessus pour changer de langue.”
- **Icône:** SVG nuage inline dans la phrase.
- **Style:** Glassmorphism subtil, blur, shadow premium, cohérent dark/light mode.
- **Responsive:** Max-width centré, padding fluide, adaptation mobile.

---

#### HTML à insérer au-dessus du lecteur

```html
<div class="player-cloud-hint" role="note" aria-live="polite">
  <svg viewBox="0 0 24 24" class="cloud-icon" aria-hidden="true">
    <path d="M3 15a4 4 0 0 1 4-4h1a4 4 0 0 1 8 0h1a4 4 0 1 1 0 8H7a4 4 0 0 1-4-4z"/>
  </svg>
  <span class="hint-text">
    Si tu vois ce <span class="highlight">nuage</span> dans le lecteur ci‑dessous,
    clique dessus pour <span class="highlight">changer de langue</span>.
  </span>
</div>
```

---

#### Contraintes et validations

- **Ne pas toucher:** aux iframes, sandbox, scripts du lecteur.
- **Visibilité:** Le bandeau doit s’afficher au-dessus de chaque lecteur, y compris si plusieurs lecteurs sont présents.
- **i18n:** Remplacer le texte par une clé i18n si disponible, ex. `t('player.cloud_hint')`.
- **Test:** Vérifier sur Movie Detail, TV Detail, et pages où le lecteur est embarqué; responsive OK; aucune redirection ou interaction perturbée.
