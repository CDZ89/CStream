Introduction to the API
More information about the API of WatchFooty

Primary use for this API is to integrate our embeds into your own applications whilst also being able to present your own live score data.

Endpoints
The API has the following endpoints:

/api/v1/matches/[sport] [?date=2025-01-01]: This endpoint allows you to get the matches available to watch for a specific sport.

/api/v1/matches/live [?date=2025-01-01]: This endpoint allows you to get the live matches available to watch.

/api/v1/matches/[sport]/popular [?date=2025-01-01]: This endpoint allows you to get the popular matches for a specific sport.

/api/v1/matches/[sport]/live [?date=2025-01-01]: This endpoint allows you to get the live matches for a specific sport.

/api/v1/matches/popular [?date=2025-01-01]: This endpoint allows you to get the popular matches across all sports.

/api/v1/matches/popular/live [?date=2025-01-01]: This endpoint allows you to get the popular live matches across all sports.

/api/v1/matches/all [?date=2025-01-01]: This endpoint allows you to get all matches across all sports.

/api/v1/matches/all/live [?date=2025-01-01]: This endpoint allows you to get all live matches across all sports.

/api/v1/matches/all/popular [?date=2025-01-01]: This endpoint allows you to get all popular matches across all sports.

/api/v1/matches/all/popular/live [?date=2025-01-01]: This endpoint allows you to get all popular live matches across all sports.

/api/v1/match/[id]: This endpoint allows you to get the details for a match.

/api/v1/match/[id]/stats: This endpoint allows you to get detailed statistics for a match including boxscore, rosters, and commentary.

/api/v1/poster/[id]: This endpoint allows you to get the poster for a match.

/api/v1/team-logo/[id]: This endpoint allows you to get the logo for a team.

/api/v1/league-logo/[id]: This endpoint allows you to get the logo for a league.

/api/v1/sports: This endpoint allows you to get the sports available to watch.

/api/v1/emotes: This endpoint allows you to get the emotes available to use in the chat.

/api/v1/top-leagues or /api/v1/top-leagues/[sport]: This endpoint allows you to get the top leagues for all sports or a specific sport.

/api/v1/top-teams or /api/v1/top-teams/[sport]: This endpoint allows you to get the top teams for all sports or a specific sport.

Rules
The API has no rate limits but we reserve the right to block abuse (please implement caching).
The embeds can be used for commercial and non-commercial purposes as long as the embeds are not altered.
Embeds are allowed to be used on other websites.
NO altering the functionality of our embeds. Violators will be forced to sit down and watch an entire WNBA game.

Matches
Using the API to get the matches available to watch for a specific sport

The matches API provides several endpoints to retrieve matches across different sports with various filtering options. All endpoints return matches for today (past 12 hours and future 12 hours) by default, unless a date parameter is specified.

Get Matches by Sport
This will return the matches for a specific sport. You can also optionally pass a date to get the matches for a specific date.

/api/v1/matches/[sport] [?date=2025-01-01]
interface Match {
  matchId: string;
  title: string;
  poster: string;
  teams: {
    home: {
      name: string;
    };
    away: {
      name: string;
    };
  };
  scores: {
    home: number;
    away: number;
  };
  status: string;
  currentMinute: string;
  currentMinuteNumber: number;
  isEvent: boolean;
  date: string;
  timestamp: number;
  league: string;
  sport: string;
  streams: {
    id: string;
    url: string;
    quality: string;
    language: string;
    isRedirect: boolean;
    nsfw: boolean;
    ads: boolean;
  }[];
};

const Matches = await fetch("https://api.watchfooty.st/api/v1/matches/football");
const MatchesData = await Matches.json() as Match[];

if (MatchesData.length > 0) {
  const Match = MatchesData[0];
  const Streams = Match.streams;
  if (Streams.length > 0) {
    const Stream = Streams[0];
    console.log(Stream); // { id: "123", url: "https://spiderembed.top/xxxx", quality: "1080p", language: "en", isRedirect: false, nsfw: false, ads: false }
  }
  console.log(Match); // { matchId: "123", title: "Arsenal vs Manchester United", poster: "https://spiderembed.top/api/match-poster?matchId=123", teams: { home: { name: "Arsenal" }, away: { name: "Manchester United" } }, isEvent: false, date: "2025-01-01", timestamp: 1714732800, league: "Premier League", sport: "football", streams: [{ id: "123", url: "https://spiderembed.top/xxxx", quality: "1080p", language: "en", isRedirect: false, nsfw: false, ads: false }] }
}

// Specifying a date
const DateString = "2025-01-01";
const Matches = await fetch(`https://api.watchfooty.st/api/v1/matches/football?date=${DateString}`);
const MatchesData = await Matches.json() as Match[];

if (MatchesData.length > 0) {
  console.log(MatchesData);
}

Top Leagues
Using the API to get the top leagues for all sports or a specific sport

The top leagues API provides endpoints to retrieve top leagues across all sports or for a specific sport. This includes league names and their aliases.

Get All Top Leagues
This will return all top leagues from all sports, including aliases.

/api/v1/top-leagues
const TopLeagues = await fetch("https://api.watchfooty.st/api/v1/top-leagues");
const TopLeaguesData = await TopLeagues.json() as string[];

if (TopLeaguesData.length > 0) {
  console.log(TopLeaguesData[0]); // "Hot Matches"
}
Copy
Get Top Leagues by Sport
This will return the top leagues for a specific sport, including aliases.

/api/v1/top-leagues/[sport]
const TopLeagues = await fetch("https://api.watchfooty.st/api/v1/top-leagues/football");
const TopLeaguesData = await TopLeagues.json() as string[];

if (TopLeaguesData.length > 0) {
  console.log(TopLeaguesData[0]); // "Hot Matches"
}
Copy
Alternative Endpoints
You can also use /api/v1/top-leagues/all to explicitly get all top leagues (same as the base endpoint).

Response Format
The response is an array of strings containing league names and their aliases:

/api/v1/top-leagues/football
[
  "Hot Matches",
  "World Cup",
  "English Premier League",
  "UEFA Champions League",
  "World Cup Qualifying",
  "FIFA World Cup Qualifying",
  "FIFA World Cup Qualifying - CONMEBOL",
  "FIFA World Cup Qualifying - Concacaf",
  "MLS",
  ...
]
Copy
Error Handling
If an invalid sport is provided, the API will return a 400 error:

Error Response
{
  "error": "Invalid sport"
}
